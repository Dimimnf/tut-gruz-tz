# Tut Gruz Frontend

Фронтенд-приложение для сервиса каталога контейнеров Tut Gruz, разработанное как Telegram Mini App.

## Возможности
- **Каталог**: Просмотр типов контейнеров (20' Стандарт, 40' HC, 40' Рефрижератор).
- **Фильтрация**: Просмотр отфильтрованных списков контейнеров.
- **Детали**: Просмотр подробной информации о контейнере и изображений.

## Требования
- Node.js (v18 или выше)
- npm или yarn

## Установка

1. **Установка зависимостей**:
   ```bash
   npm install
   ```

2. **Настройка окружения**:
   Создайте файл `.env` в корневой директории (или в родительской директории, как настроено в конфиге) со следующими переменными:
   ```env
   VITE_API=https://your-api-url.com
   VITE_MODE=DEV # или PROD
   ```

## Разработка

Для запуска сервера разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

### Имитация Telegram Mini App
В режиме разработки (`VITE_MODE=DEV`), приложение имитирует среду Telegram с тестовыми данными пользователя.

## Сборка

Для сборки проекта в продакшн:
```bash
npm run build
```

## Docker

Для запуска приложения в контейнере:

1. **Сборка образа**:
   ```bash
   docker build -t tut-gruz-front .
   ```

2. **Запуск контейнера**:
   ```bash
   docker run -p 5173:5173 -v %cd%/src:/app/src -v /app/node_modules tut-gruz-front
   ```
   *(Для Linux/Mac замените `%cd%` на `$(pwd)`)*

Приложение будет доступно по адресу `http://localhost:5173`.
Контейнер автоматически запускает команду `npm run dev`.
